Available Courses

Introduction to Programming (Kids)

This course is designed for children aged 8-14 years old. It introduces them to the basics of programming using fun and interactive tools such as Scratch. The goal is to ignite curiosity and foster a love for coding from an early age.

Html-CSS Programming (Beginner)

Learn the fundamentals of Html-CSS programming. This course is perfect for absolute beginners and will help you understand basic concepts such as variables, loops, and conditionals. Html-CSS is a great language to start with and is widely used in various fields like web development.

Web Development with JavaScript

This course focuses on JavaScript, one of the most powerful and versatile languages for web development. You will learn how to create interactive websites, and work with the Document Object Model (DOM) to manipulate HTML and CSS. A great course for those interested in front-end development.

Advanced Programming Techniques (Adults)

This advanced course is aimed at individuals who already have a basic understanding of programming. We dive deep into topics like algorithms, data structures, object-oriented programming, and more. A great choice for those who wish to pursue a career in software development or expand their programming skills.
